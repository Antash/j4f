Написать программу, которая реализует функцию поиска файлов на локальных дисках персонального компьютера.
 
Создаваемая программа должна удовлетворять следующим требования:
-Возможность указания параметров для поиска файлов (сделать разумный выбор по своему усмотрению)
-Возможность запуска нескольких асинхронных копий процесса поиска и отображения общего результата в одном общем окне.
-Возможность остановки процессов, просмотра их статуса, отображения количества найденных файлов (например).
-Возможность расширения параметров поиска с помощи подключаемых модулей (Plugins)
	-Реализовать модуль поиска по тегу внутри XML
	-Реализовать модуль поиска класса/интерфейса среди дотнетовских сборок.
-Устойчивость к ошибкам.
-Интерфейс – любой WPF, WinForms, Web
-Язык реализации – C#
